# Four Cylinder Engine in SolidWorks

## Project Overview
This project presents the design and assembly of a Four Cylinder Internal Combustion Engine using SolidWorks. It demonstrates CAD modeling techniques and assembly design principles.

## Components Designed
- Piston
- Piston Ring
- Crankshaft
- Connecting Rod
- Connecting Rod Cap
- Piston Pin
- Engine Assembly

## Software Used
- SolidWorks

## Skills Demonstrated
- 3D Part Modeling
- Assembly Design
- Engineering Drawings
- Mechanical Component Design
- CAD Documentation
